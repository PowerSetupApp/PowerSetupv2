# PowerSetup – Übersicht

## Vision & Ziel

PowerSetup ist eine mobile-first Web-App, die **absolute Camping- und Elektrik-Anfänger** dabei unterstützt, ein **vollständiges, sicheres und normnahes Strom-Setup** für Fahrzeuge zu planen.

**Ergebnis für den Nutzer:**
- Vollständige Einkaufsliste (Affiliate)
- PDF-Schaltplan (gegen Credits)
- Warnhinweise & Haftungsausschlüsse

---

## Zielgruppe

- Absolute Anfänger ohne Elektrik-Kenntnisse
- Mobile-first (80–90 % Nutzung)
- EU-Fokus (Normen, Sprache, Haftung)

---

## Supported Setups (v1)

### Fahrzeugtypen
- Campervan
- Wohnmobil
- Wohnwagen
- Boot
- Offroad / Expeditionsfahrzeuge

### Energiequellen
- Bordbatterien (AGM, Gel, Lithium)
- Solar
- Lichtmaschine (Ladebooster)
- Landstrom (230V)
- Generator

### Verbraucher
- Beleuchtung
- Kühlgeräte
- Heizung
- Wasserpumpe
- Ladegeräte (USB, 12V)
- Wechselrichter / 230V-Verbraucher

---

## User Flow (High-Level)

```
1. Landing Page (Erklärung + CTA)
   ↓
2. Multi-Step Formular (adaptiv)
   ↓
3. Ergebnis-Seite:
   - Produktempfehlungen (kostenlos)
   - Verbrauchsübersicht (kostenlos)
   - CTA: „Schaltplan als PDF generieren"
   ↓
4. Credit-Kauf (PayPal)
   ↓
5. PDF-Generierung + Download
   ↓
6. Ergebnis über persistente URL (90 Tage gültig)
```

---

## MVP Scope

### ✅ Im MVP enthalten
- Multi-Step Formular
- Produktempfehlungen (kostenlos)
- PDF-Generierung (kostenpflichtig)
- PayPal-Zahlung mit Credit-Paketen
- Persistente URLs mit Versionierung
- Simple Share-Buttons
- Alle Fahrzeugtypen inkl. Boot

### ❌ Nicht im MVP
- User Accounts
- Manuelle Schaltplan-Editoren
- E-Mail-Versand
- Community-Features
