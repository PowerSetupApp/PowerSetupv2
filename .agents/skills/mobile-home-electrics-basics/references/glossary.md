## Formula symbols used throughout this skill


| Symbol  | Meaning                  | Unit     |
| ------- | ------------------------ | -------- |
| `U`     | voltage                  | V        |
| `I`     | current                  | A        |
| `P`     | power                    | W        |
| `E`     | energy                   | Wh       |
| `R`     | resistance               | Ω        |
| `ρ`     | resistivity              | Ω·mm²/m  |
| `L`     | one-way cable length     | m        |
| `A`     | conductor cross-section  | mm²      |
| `η`     | efficiency (0 – 1)       | –        |
| `PSH`   | peak sun hours per day   | h/day    |
| `DoD`   | depth of discharge       | fraction |
| `SoC`   | state of charge          | fraction |
| `C_nom` | nominal battery capacity | Ah or Wh |
| `cos φ` | power factor (AC)        | –        |
