# PowerSetup – Projekt-Übersicht

## Vision
Mobile-first Web-App für **Camping-/Elektrik-Anfänger** zur Planung eines vollständigen, sicheren Strom-Setups für Fahrzeuge.

## Zielgruppe
- Absolute Anfänger ohne Elektrik-Kenntnisse
- Mobile-first (80-90% Nutzung)
- EU-Fokus (Normen, Sprache, Haftung)

## Ergebnis für Nutzer
1. **Kostenlos**: Produktempfehlungen (Affiliate), Verbrauchsübersicht
2. **Kostenpflichtig**: PDF-Schaltplan (gegen Credits)
3. Warnhinweise & Haftungsausschlüsse

## Supported Setups (v1)

### Fahrzeugtypen
Campervan, Wohnmobil, Wohnwagen, Boot, Offroad/Expedition

### Energiequellen
Bordbatterie (AGM/Gel/LiFePO4), Solar, Lichtmaschine, Landstrom 230V, Generator

### Verbraucher
Beleuchtung, Kühlgeräte, Heizung, Wasserpumpe, USB/12V-Ladegeräte, Wechselrichter/230V

## User Flow
```
Landing Page → Multi-Step Formular → Ergebnis (kostenlos)
→ Credit-Kauf (PayPal) → PDF-Generierung → Persistente URL (90 Tage)
```

## MVP Scope
✅ Multi-Step Formular, Produktempfehlungen, PDF-Generierung, PayPal, Persistente URLs
❌ User Accounts, manuelle Editoren, E-Mail-Versand, Community

## Recht & Sicherheit
- Klarer Haftungsausschluss
- Warnhinweise bei 230V
- Empfehlung Fachbetrieb
- DSGVO: Keine personenbezogenen Daten, Auto-Löschung 90 Tage

## Internationalisierung
- DE / EN (i18n)
- Texte KI-seitig sprachabhängig

## Erfolgskriterien
- Formular Completion Rate > 70%
- Conversion zu Schaltplan > 10%
- Affiliate CTR > 15%
